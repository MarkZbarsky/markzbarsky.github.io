Wind Turbine Sensor Integration

Research Project • January - May 2024

Overview

During my research collaboration with Polymathic Innovations, I tackled the complex challenge of instrumenting a prototype Savonius vertical-axis wind turbine for comprehensive performance analysis. The project required developing a robust data acquisition system capable of interfacing with multiple sensor types distributed throughout the turbine structure. I architected the solution using C++ for low-level sensor communication and Python for data processing and export functionality.

System Architecture

The system continuously monitored critical performance parameters, automatically logging data to structured Excel formats for analysis. Through systematic evaluation of the collected metrics, I identified several optimization opportunities that our team implemented iteratively. These data-driven modifications ultimately resulted in a 15% improvement in overall turbine efficiency.

Development Process

Initial Assessment

The project began with evaluating the existing turbine prototype and identifying key performance metrics that needed monitoring. We determined that RPM, torque, wind speed, and power output were the critical parameters to track.

Sensor Selection & Integration

I researched and selected appropriate sensors for each metric, including hall effect sensors for RPM measurement, strain gauges for torque, and anemometers for wind speed. Each sensor required custom mounting solutions to ensure accurate readings without affecting turbine performance.

Data Collection System

I designed a microcontroller-based data acquisition system using an Arduino Mega as the central hub. This system collected data from all sensors at configurable intervals and transmitted it to a Raspberry Pi for processing and storage.

Software Development

I wrote C++ firmware for the Arduino to handle real-time sensor reading and data transmission. On the Raspberry Pi, I developed Python scripts to process incoming data, perform calculations, and export results to spreadsheets for analysis.

Optimization & Testing

Using the collected data, our team identified inefficiencies in the turbine design. We made iterative improvements to blade shape, bearing selection, and overall structure, testing each change against baseline performance metrics.

Technical Challenges

Environmental Protection

Electronic components needed protection from outdoor weather conditions.

Solution: I built simple enclosures to protect the sensors and data collection hardware from rain and dust exposure.

Data Processing

Raw sensor readings needed processing to identify meaningful trends and patterns.

Solution: I wrote code to calculate averages and standard deviations from the sensor data, helping to identify consistent performance patterns and outliers in the measurements.

Results and Impact

The project demonstrated both the practical challenges of renewable energy instrumentation and the significant impact that careful monitoring and analysis can have on system performance in urban wind applications. The sensor integration and subsequent optimization resulted in a 15% improvement in turbine efficiency. The data collection system continues to provide valuable insights for ongoing development.

The project has demonstrated the viability of Savonius VAWTs for urban environments and has led to additional funding for expanded research. This work contributed to the broader understanding of small-scale wind energy systems and their potential applications in distributed power generation.
