Wind Turbine Sensor Integration

Research Project - January - May 2024

Overview

As part of my research with Polymathic Innovations, I engineered a data collection system for a prototype Savonius vertical-axis wind turbine (VAWT). I developed custom software using C++ and Python to interface with various sensors installed throughout the turbine structure. This system continuously monitored performance metrics and published data to Excel spreadsheets for detailed analysis. Through iterative optimization based on the collected data, our team successfully improved the turbine's efficiency by 15%.

Development Process

Initial Assessment

The project began with evaluating the existing turbine prototype and identifying key performance metrics that needed monitoring. We determined that RPM, torque, wind speed, and power output were the critical parameters to track.

Sensor Selection & Integration

I researched and selected appropriate sensors for each metric, including hall effect sensors for RPM measurement, strain gauges for torque, and anemometers for wind speed. Each sensor required custom mounting solutions to ensure accurate readings without affecting turbine performance.

Data Collection System

I designed a microcontroller-based data acquisition system using an Arduino Mega as the central hub. This system collected data from all sensors at configurable intervals and transmitted it to a Raspberry Pi for processing and storage.

Software Development

I wrote C++ firmware for the Arduino to handle real-time sensor reading and data transmission. On the Raspberry Pi, I developed Python scripts to process incoming data, perform calculations, and export results to spreadsheets for analysis.

Optimization & Testing

Using the collected data, our team identified inefficiencies in the turbine design. We made iterative improvements to blade shape, bearing selection, and overall structure, testing each change against baseline performance metrics.

Technologies Used

Arduino Mega Raspberry Pi C++ Python Sensors (Hall Effect, Strain Gauges) Excel

Challenges & Solutions

Environmental Durability

Sensors needed to withstand outdoor conditions including rain, temperature variations, and dust.

Solution: I designed custom weatherproof enclosures for all electronic components and used marine-grade connectors for all external wiring.

Power Management

The monitoring system needed to operate continuously without external power sources.

Solution: I implemented a solar-powered battery system with power management circuitry to ensure continuous operation, even during cloudy periods.

Data Reliability

Initial sensor readings showed inconsistencies due to vibration and electromagnetic interference.

Solution: I added signal filtering algorithms in the firmware and implemented redundant sensors for critical measurements to ensure data accuracy.

Results

The sensor integration and subsequent optimization resulted in a 15% improvement in turbine efficiency. The data collection system continues to provide valuable insights for ongoing development. The project has demonstrated the viability of Savonius VAWTs for urban environments and has led to additional funding for expanded research.
